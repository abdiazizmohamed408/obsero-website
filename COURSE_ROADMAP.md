## Course Quality Standards

### Content Requirements
- [ ] All facts cited with hyperlinks to primary sources
- [ ] Sources from government (.gov), official bodies, reputable legal sources
- [ ] Last updated date displayed
- [ ] Jurisdiction clearly marked (US/CA/EU)
- [ ] Legal disclaimer included

### Interactive Elements
- [ ] Minimum 3 knowledge checks per module
- [ ] 1 simulation/scenario per module
- [ ] Final assessment (80% pass rate)
- [ ] Certificate of completion

### Design Standards
- [ ] Clean, modern UI (matches Obsero brand)
- [ ] Mobile responsive
- [ ] Accessible (WCAG 2.1 AA)
- [ ] Progress indicator
- [ ] Estimated time per module

### Technical
- [ ] SCORM 1.2 compliant
- [ ] Works offline
- [ ] Multi-language ready (EN/FR/ES structure)
